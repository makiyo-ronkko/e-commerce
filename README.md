# E-commerce app

## About the app

I have built the e-commerce app with Vanilla Javascript, Node.js and Express following Udemy's course 'The Modern Javascript Bootcamp Course (2020)' instructed by Stephen Grider.

## Features

- Rest API
- Repository approach
- Cookie based authentication
- Hashing password
- Upload images
- Shopping cart
